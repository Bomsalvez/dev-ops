## 🚀 COMO USAR (PASSO A PASSO)

### Opção 1: Baixar ZIP (Mais fácil)

```bash
# 1. Baixe o arquivo: aulas_devops_completas.zip

# 2. Extraia (duplo clique no arquivo ou use o comando)
unzip aulas_devops_completas.zip

# 3. Entre na pasta
cd aulas

# 4. Comece pela primeira aula
```

### Opção 2: Baixar TAR.GZ (Linux/Mac)

```bash
# 1. Baixe: aulas_devops_completas.tar.gz

# 2. Extraia
tar -xzf aulas_devops_completas.tar.gz

# 3. Entre na pasta
cd aulas
```
