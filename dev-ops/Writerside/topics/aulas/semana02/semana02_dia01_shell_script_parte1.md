# Shell Script - Fundamentos

## 📚 Tema: Programação em Bash

### Conceitos Principais
- Shebang (#!/bin/bash)
- Variáveis
- Entrada do usuário
- Argumentos ($1, $2, $@)

---

## 💪 Prática

```bash
#!/bin/bash
NOME=$1
echo "Olá, $NOME!"
echo "Total de args: $#"
```
